# Notification Dashboard UI

## Overview

This project is developed using React.js and Material UI. It displays notifications in a simple and user-friendly interface.

## Features

- View all notifications.
- View priority notifications.
- Responsive design for desktop and mobile.
- Logging middleware integration.
- Error handling for API requests.

## Pages

### All Notifications
Displays all notifications received from the API.

### Priority Notifications
Displays the top priority notifications based on notification type.

## Design

- Material UI components
- Simple navigation bar
- Notification cards
- Responsive layout
- Clean and easy-to-read interface

## Logging

Logging middleware records:

- Page loading
- API requests
- Successful operations
- Error messages

## Conclusion

The application provides a clean, responsive, and simple notification dashboard with proper logging and error handling.
